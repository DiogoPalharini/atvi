# Automanager

Micro-serviço para cadastro dos dados do cliente.

## Pré-requisitos

- Java 17 ou superior
- Maven (ou use o Maven Wrapper do projeto)

## Como executar

```bash
cd automanager
mvn spring-boot:run
```

Ou usando o Maven Wrapper:

**Windows:**
```bash
cd automanager
.\mvnw.cmd spring-boot:run
```

**Linux/Mac:**
```bash
cd automanager
./mvnw spring-boot:run
```

## Acessar a aplicação

Após iniciar, a aplicação estará disponível em: http://localhost:8080
